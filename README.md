# Sketchers-Sales-Analysis
This Power BI dashboard delivers a comprehensive analysis of Skechers’ sales performance, offering insights across time periods, regions, product categories, and retail channels. It enables users to identify key trends, evaluate profitable segments, and explore high-performing markets to support data-driven strategic decision-making.

The primary objective of this dashboard is to monitor sales performance, uncover growth patterns, and highlight high-value regions and high-demand products. These insights help stakeholders make informed business decisions and optimize sales strategies.

Tools & Technologies Used:
Power BI – Interactive dashboard development
DAX – Calculated measures and performance metrics
Excel/CSV – Structured data source
Power Query – Data cleaning, preprocessing, and transformation
GitHub – Version control, documentation, and project collaboration
